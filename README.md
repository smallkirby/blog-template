# Blog Template and Relative Tools  for HatenaBlog
ブログ書くときは、ずっと丹精込めてHTMLを直打ちしていたんですが、流石にめんどすぎるので辞めます。  
自分用に調整されたMarkdown to HTMLコンバータです。  

## 機能
- 現行のCSS似合わせたヘッダ変換
  - 最大`<h>`タグは`<h3>`
- 見出し自動インデックス
- コードブロック対応
  - ファイル名に応じてPrettyPrint
- ヘッダとフッダ
